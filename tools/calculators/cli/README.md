# Zc Calculator CLI

Interactive command-line tool for calculating Cognitive Impedance.

## Usage

```bash
# Interactive mode
python zc_cli.py --interactive

# Direct calculation
python zc_cli.py --vgen 25 --bsocial 30

# From metrics
python zc_cli.py --slack 150 --notion 20 --ai 30 --team-size 10
```

Zero external dependencies required.
