## Migration Guides

### From v2.0 to v3.0

**What changed:**
- Repository structure reorganized
- Documentation split into docs/ folder
- Tools added to tools/ folder

**Action required:**
1. Re-read README.md (completely rewritten)
2. Use new templates in tools/templates/
3. Start with Quick Start Guide
4. Ignore old PROTOCOL-001-CORE.md (merged into other docs)

**What still works:**
- Core Zc formula unchanged
- Mode definitions unchanged
- Thresholds unchanged (0.7, 1.0)

### From v1.0 to v3.0

**Recommend:** Start fresh with v3.0. Conceptual foundation is the same, but everything else is rebuilt.
